# Chart types & variations

_Generated by `scripts/gen-panel-docs.py` — do not edit by hand._

Two layers of chart definitions:

- **Base presets** (`libs/common-lib/panels/`) — pre-styled panels per category,
  `g.common.panels.<category>.<preset>(title, targets, description)`.
- **Variation boards** (`reference/panels/`) — a tabbed board per Grafana panel
  type showing its options (rendered in the Panel Reference folder).
